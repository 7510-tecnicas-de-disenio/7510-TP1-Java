# Simple Logic Interpreter [![Build Status](https://travis-ci.org/abrden/7510-TP1-Java.svg?branch=master)](https://travis-ci.org/abrden/7510-TP1-Java)

A Java library designed to serve as a logic interpreter.

- Java 8 or greater
- Maven 3.3.9
